## 格式规范

### 标题

- 章节标题（`##` `###`）直接写名称，**不加数字编号**
    - 正确：`## 核心概念`
    - 错误：`## 1. 核心概念`
- 加编号后增删章节需手动重排序号，维护成本高，所以避免使用

### 目录

- **不主动添加目录**（`## 目录` / Table of Contents）
- 用户明确要求时再加
- 原因：Typora、JetBrains IDE 等工具自带文档大纲导航，目录冗余

### 表格

分隔符固定写 `| --- |`（三个连字符，左右各一个空格），内容列同样保持紧凑：

```markdown
| 列A | 列B | 列C |
| --- | --- | --- |
| 内容 | 内容 | 内容 |
```

不要用多余的连字符或空格撑宽列：

```markdown
|  列A  |  列B   |
| ----- | ------ |
|  内容  |  内容   |
```
